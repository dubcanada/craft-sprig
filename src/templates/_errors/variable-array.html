The variable `{{ name }}` is an array and cannot be passed into a Sprig component. Instead of passing an array, consider passing a comma-separated string. --- Before: {{ '{{' }} sprig('_components/my-component', { '{{ name }}': {{ name }} }) {{ '}}' }} --- After: {{ '{{' }} sprig('_components/my-component', { '{{ name }}': {{ name }} | join(',') }) {{ '}}' }} Then in your component you could split the string. {{ '{%' }} set {{ name }} = {{ name }} | split(',') {{ '%}' }}
